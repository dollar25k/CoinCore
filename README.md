# Installation

To install CoinCore, follow these steps:

1. Clone the repository using `git clone https://github.com/[username]/CoinCore.git`
2. Install the required dependencies using `pip install -r requirements.txt`
3. Create a new PostgreSQL database and Redis instance
4. Configure the `config.py` file to point to the newly created database and Redis instance
5. Run the application using `python app.py`

# Configuration

CoinCore's configuration options can be found in the `config.py` file. The following options are available:

* `DB_HOST`: The hostname of the PostgreSQL database instance
* `DB_PORT`: The port number of the PostgreSQL database instance
* `DB_USER`: The username for the PostgreSQL database instance
* `DB_PASSWORD`: The password for the PostgreSQL database instance
* `REDIS_HOST`: The hostname of the Redis instance
* `REDIS_PORT`: The port number of the Redis instance
